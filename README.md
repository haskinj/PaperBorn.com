# PaperBorn

> Making research playable.

PaperBorn is an experimental project exploring what happens when academic papers are transformed into characters, entities, artifacts, and worlds.

A research paper is treated not just as information, but as:
- structure
- behavior
- symbolism
- tension
- embodiment
- mythology

Each PaperBorn entity emerges from the underlying paper itself:
its mathematics, language, themes, abstractions, citations, uncertainty, and conceptual geometry.
